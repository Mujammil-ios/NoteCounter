Project: Note Counter
==================================

Money Counter is a simple application to count your currency of various denominations. 

Introduction
------------

Money Counter is a simple application to count your currency of various denominations. Handy tool to count your precious money. Useful at Billing Centers, Shopping Malls, Banking Counters etc. An Essential Everyday Utility.

Getting Started
---------------

1. All Currency notes are available (Include 2000 and 200 new currency note.)
2. Simple to use.
3. App is designed using material design technology.
4. Specially designed for Indian users

Key-Features
----

-Allows you to retain the denomination values.

-Calculated values and results can be shared to social media, mail, messages and other sharing apps.

-Currency formats according to language preferences.

-Options to choose and remove denominations.

-Automatic and accurate calculation.

-Professional user interface.

-Pleasant presentation.
